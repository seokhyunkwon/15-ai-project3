# 로그인 필수 적용 패치

## 수정 내용

- URL 파라미터 `?login_user=...` 만으로 로그인되던 우회 문제 제거
- 로그인/회원가입 링크에 `target="_self"` 적용하여 새 창/새 탭 대신 현재 화면에서 이동
- 로그인하지 않은 경우 검색/추천/숙소/식당/찜/데이터 상태 기능 화면을 렌더링하지 않음
- 상단 링크가 환경에 따라 안 먹을 때를 대비해 로그인 필수 안내 영역에 Streamlit 버튼 추가

## 적용 방법

1. 이 폴더의 `app.py`를 기존 프로젝트의 `app.py`에 덮어쓰기
2. 이전 지역 수정 패치를 아직 적용하지 않았다면 `database.py`, `collector.py`, `schema.sql`도 같이 덮어쓰기
3. Streamlit 재실행

```powershell
Ctrl + C
streamlit run app.py
```

## 확인 방법

- 로그아웃 상태: 검색 폼이 보이지 않아야 함
- 로그아웃 상태: 로그인 안내와 로그인/회원가입 버튼만 보임
- 로그인 성공 후: 검색 폼과 추천 기능 사용 가능
- 주소창에 `?login_user=demo`를 직접 붙여도 로그인되면 안 됨
